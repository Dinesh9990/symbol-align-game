# Symbol Align Game

A puzzle game built with **React and Vite** where players align matching symbols in a 4×4 grid using strategic swaps.

## 🎮 Gameplay

* Click a symbol to select it
* Use arrow buttons to swap with neighboring symbols
* Align 4 identical symbols in a row or column
* Align all symbol types to win the game
* Fewer moves = higher score

## 🚀 Features

* 4×4 dynamic puzzle grid
* Randomized symbol placement
* Move-based scoring system
* Restart game functionality
* Interactive arrow controls

## 🛠 Technologies Used

* React
* Vite
* JavaScript
* CSS
* Material UI Icons

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/symbol-align-game.git
```

Navigate into the project folder:

```bash
cd symbol-align-game
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the browser:

```
http://localhost:5173
```
